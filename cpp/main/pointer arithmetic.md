# pointer arithmetic

adding 1 to a pointer variable increases the value by the number of bytes of the type

say, if double is 8 bytes -> adding 1 adds 8 to the pointer value

e.g. 
`int stacks[3]; // stacks[0] == *(stacks + 1)`
